Interface Usage

Hover over the eyes to see a demonstration

Moving

Click and drag an empty space to move the viewing window.

Zooming

Use the scroll wheel to expand the graph in and out (maintains apparent size).

While scrolling, hold the ctrl key to zoom instead (maintains relative size).

Searching

Enter text into the search bar to highlight all nodes whose name contains that query.

If only one node is found, that node will be brought to the center.

Use the button to the left of the search bar to switch between searching for nodes or edges.

To search for multiple nodes or edges, enter each name separated by a comma.

All terms except for the last will require an exact match.

Click away from the search bar to clear the current search.

Organization

Click and drag a node to change its position.

Click the Organize button to optimize the arrangement of nodes.

Use the Physics button to enable and disable a live physics simulation of the graph.

If node clustering is available, use the + and - keys to expand and collapse the clusters.

Node Connections

Left-click a node to highlight its immediately adjacent neighbors.

Left-click a second node to highlight the shortest path between the two, if one exists.

Information

Right-click a node or edge to view its data properties.

Left-click two adjacent nodes to show the data properties of all edges between them.
